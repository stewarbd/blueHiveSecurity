# BlueHive Security

Website for BlueHive Security — community-focused cybersecurity work.

## Local development

Just open `index.html` in a browser. It's a single static file with no build process.

## Deployment

Push to GitHub and enable Pages in the repo settings. Site will be live at `https://yourusername.github.io/bluehive-security` (or whatever you name the repo).

## Updating content

Edit `index.html` directly. All styling is inline, all copy is in the HTML.
